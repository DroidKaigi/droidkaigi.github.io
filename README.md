## ローカルでの確認方法

### 事前準備
1. rubyのインストール(ver 1.9.3 or 2.0.0)
1. bundlerのインストール
1. github用のgemをインストール  
`bundle install --path vendor/bundle`

### 確認
1. jekyllを起動  
`bundle exec jekyll serve`
1. http://127.0.0.1:4000/ にアクセス
